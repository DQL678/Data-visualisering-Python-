# Data-visualisering-Python-
Trello: https://trello.com/b/Hhcw8009/datavisualisering-python

Link til API: https://www.freepublicapis.com/exchange-rate-api 

Format: JSON.

**Case**:
I dette projekt vil der arbejdes med datavisualisering af valutakurser. Formålet med projektet er at hente data fra en offentlig API og præsentere dem på en overskuelig måde, så brugeren kan få et bedre indblik i, hvad kursen er for forskellige valutakurser og, hvordan valutakurser udvikler sig over tid. Vi ønsker at gøre det muligt for brugeren at sammenligne forskellige valutaer med hinanden, se kursen på enkelte valutaaer, se, hvornår det er mest fordelagtigt at veksle penge, og se graffer over forskellige valutaers udvikling over tid sammenlignet med andre valutaer.
